🚀 Project: DSA Learning Platform

Description:
Developed a web-based platform to help users learn and track Data Structures & Algorithms efficiently through structured topics, problem tracking, and performance analytics.

🔧 Key Features
📚 Topic-Based Learning: Organized DSA topics like Arrays, Strings, Trees, Graphs, etc.
✅ Problem Tracker: Mark problems as solved/unsolved with difficulty levels
📝 Notes System: Add personal notes, approaches, and mistakes for each problem
📊 Dashboard Analytics: Track total solved problems and topic-wise progress
🔍 Search & Filters: Filter problems by difficulty, topic, and status
🔁 Revision System: Mark problems for revision and track learning consistency
